# 🎱 LuckyBall (럭키볼) - 행운의 공 추첨기

순수 **HTML / CSS / JavaScript** 만을 사용하여 구현된 웹 기반 당첨자 추첨 서비스입니다. 
외부 라이브러리 없이 자체 구현한 2D 물리 엔진(Canvas)과 다크모드 기반의 세련된 UI를 통해 각종 이벤트, 회식, 모임 등에서 누구나 쉽고 재미있게 활용할 수 있습니다.

## ✨ 주요 기능

- **두 가지 추첨 모드**
  - **🏆 당첨자 뽑기**: 로또 추첨기처럼 공이 튀어오르며 한 명씩 차례대로 당첨자를 선정합니다.
  - **📊 순서 정하기**: 참가자 전체의 순번을 폭죽/분수 연출과 함께 한 번에 빠르게 정합니다.
  
- **자연스러운 애니메이션 및 연출**
  - Canvas API를 활용한 볼 간의 충돌 및 중력 물리 효과 적용
  - Web Audio API를 이용한 외부 에셋 없는 내장형 효과음(사운드) 구현
  - 추첨 완료 시 화려한 폭죽(Confetti) 파티클 애니메이션

- **편리한 설정 및 UI**
  - **직접 입력 지원**: 참가자 리스트를 이름, 혹은 숫자 범위로 쉽게 입력
  - **프리셋 저장 기능**: 자주 쓰는 참가자 명단(동아리, 팀원 등)을 `localStorage`에 저장하고 언제든 불러오기 가능
  - **다크모드(Dark Mode) 디자인**: 어두운 배경과 쨍한 비비드 컬러 공들의 조합으로 직관적이면서 세련된 사용자 경험(UX) 제공
  - 화면 크기에 유동적으로 반응하는 모바일 대응(Responsive) 환경

## 🛠 기술 스택

- **Frontend**: HTML5, CSS3 (CSS Variables, Flex/Grid Layout), Vanilla JavaScript (ES6)
- **APIs**: 
  - `Canvas API` (물리 연산/렌더링)
  - `Web Audio API` (사운드 합성)
  - `localStorage` (프리셋 데이터 유지)
  - `requestAnimationFrame` (부드러운 루프 애니메이션)

## 🚀 사용/실행 방법

별도의 빌드 시스템이나 웹 서버 인프라 없이, 다운로드 후 즉시 사용 가능합니다.

1. 본 레포지토리의 파일들을 클론(Clone) 하거나 다운로드합니다.
2. 다운로드 된 폴더 내의 `index.html` 파일을 크롬(Chrome), 사파리(Safari), 엣지(Edge) 등의 모던 웹 브라우저로 엽니다.
3. 참가자를 입력하고 모드를 선택한 뒤 무작위 추첨을 시작하세요!
